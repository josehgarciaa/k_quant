Spectral Solvers¶

class k_quant.spectral_solvers.kpm.Density(syst, bounds, kernel='jackson', X=None)¶

The spectral density of an operator \(X\).

The spectral density is defined as \(X(E) = {\rm Tr}[ \hat{X} \delta(H-E) ]\) and quantifies how much of the operator X will be measured at a given en energy .

In this module \(\delta(H-E)\) is computed using the kernel polynomial method (KPM).

Parameters:

  • syst (k_quant.system) – A properly initialized system.

  • bounds (str) – The energy bounds (in the same units as the Hamiltonian) used to rescaled the hamiltonian spectrum to the (-1,1) interval.

  • kernel (str, optional) – A string indicating the kernel to be used for the regularization. The options are: “jackson, lorentz”. For any other option will use default: Jackson

  • X (k_quant.operator, optional) – The operator used to compute the density. When none submitted will use identity

Note

Using this module, will result in rescaling the hamiltonian operator defined in system. For returning it to the original one, please call the method self.OriginalHam().
