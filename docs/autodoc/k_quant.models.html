class k_quant.models.wannier.WannierSystem(**kwargs)¶

Bases: object

A class that handles Wannier-based Tight binding models in different Wannier

Keyword Arguments:

The arguments are presented in order of preference, i.e, wout will be used over label.

label (str): The label of a Wannier90 calculation. This label will be used to look for the label.uc, label_hr.dat and label.xyz files. The positions and primitive lattice vectors in label.xyz and label.uc respectively, should be in the same units

Convention It is important to highlight that in our label.xyz we only include the positions of the Wannier center and not the atoms
